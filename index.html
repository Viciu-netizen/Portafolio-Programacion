  • Codigo CSS para diferentes dispositivos electronicos.

    • SMARTPHONE 0PX-600PX
    • TABLET 600 PX - 1000PX
    • MONITOR-TY 1000PX - ADELANTE
    
                        body {
                            background-color: blue;
                        }
                
                        @media (max-width: 600px) {
                            body {
                                background-color: red;
                            }
                        }
                
                        @media (min-width: 600px) and (max-width: 1000px) {
                            body {
                                background-color: green;
                            }
                        }
                
                        .estoy-en-pc {
                            display: none;
                            @media (min-width: 1000px) {
                                display: inherit;
                            }
                        }
                
                        .estoy-en-tablet {
                            display: none;
                            @media (min-width: 600px) and (max-width: 1000px) {
                                display: inherit;
                            }
                        }
                
                        .estoy-en-movil {
                            display: none;
                            @media (max-width: 600px) {
                                display: inherit;
                            }
                        }
